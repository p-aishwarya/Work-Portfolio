About Web Scraping


Initial thoughts on web scraping as a work would be trivial, easy, and unimportant. However, I have realised through the 4 weeks of internship at Multiverse that it is one of the most important steps in the Data science pipeline which most of us turn a blind eye to.

Structured data is of paramount importance when it comes to data analytics but more than 70% of data present online is unstructured and hence is of little use. However, Web scraping allows us to “legally” collect information from thousands of websites and structure the data into desirable format which eases further steps of the Data science pipeline.

About AWS Cloud Platform


There exists a number of ways to store data in cloud but I have used Amazon S3 to store files because it provides object (file) storage through a web interface. It's built to store, protect and retrieve data from “buckets” at any time from anywhere on any device. A user creates a bucket, and the bucket stores objects in the cloud.

AWS Lambda is a serverless compute service that runs your code in response to events and automatically manages the underlying compute resources for you.

Amazon API Gateway is a fully managed service that makes it easy for developers to create, publish, maintain, monitor, and secure APIs at any scale. APIs act as the "front door" for applications to access data, business logic, or functionality from your backend services.

About BERT & FuzzyWuzzy Model


The reason for choosing BERT algorithm is that it was developed by Google and it is the most lucrative NLP model available for context matching. Applying BERT, Google is able to understand the relationship between words in a query and present the closest matching result to the searcher.

In the example below, the context of the word “bank” is different in both sentences.Google needs to process the context of the words before the word “bank” in the first query and after the word “bank” in the second query. This is what bidirectional query processing is. Google can process a query both ways depending on the context.

FuzzyWuzzy algorithm is mainly used for string matching and hence we used it for matching the job titles. FuzzyWuzzy also has more powerful functions to help with matching strings in more complex situations. The partial ratio() function allows us to perform substring matching.
